# Cartes Custom Yu-Gi-Oh! et Decks Version Manga et Anime pour Project Ignis EDOPro

**Créateurs des cartes custom: Kal/Angel**

Bienvenue dans ce projet de cartes custom Yu-Gi-Oh! pour EDOPro/YGOPro.  
Ce dépôt contient des decks, puzzles, images et scripts pour utiliser ces cartes dans le jeu.

Ce projet propose des cartes personnalisées uniques pour enrichir votre expérience.  
Il contient : decks (.ydk) prêts à l’emploi, puzzles (scénarios solo), images de cartes pour remplacer ou ajouter aux cartes existantes, et scripts/bases de données pour le fonctionnement des cartes.

## Installation

**Decks** : copier les fichiers `.ydk` dans `EDOPro/deck/`.  
**Puzzles** : copier les fichiers dans `EDOPro/puzzles/`.  
**Images** : copier dans `EDOPro/pics/` ou `EDOPro/pics/custom/`.  
**Scripts (.lua)** : copier dans `EDOPro/script/`.  
**Bases de données (.cdb)** : copier dans `EDOPro/expansions/`.  
**Configurations (optionnel)** : copier dans `EDOPro/config/`.  

> Toujours **copier-coller**, ne pas déplacer l’original pour éviter toute perte.
